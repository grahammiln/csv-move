# csv-move

Tool to move folders listed in a CSV file.

## Test the move

Pipe in the CSV file without additional flags to test the move:

    ./csv-move < input.csv

## Perform the move

Pipe in the CSV file with a `-move` flag to perform the move:

    ./csv-move -move < input.csv
